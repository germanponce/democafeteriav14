Fast Invoicing

Helps to generate an invoice on the website page by jut adding a PoS reference.
Features

Fast Invoicing module allows to invoice every order registered on the PoS module. Customers with a PoS order reference have the posibility to fetch for a single order so it can be invoiced from the website page. It can be done by customers that are either registered in the system or are not registered.

Adding PoS reference

On the top left side of the website the Fast Invoicing menu brings a search box which look for the PoS reference that is added in it. It gets the order associated with the PoS order. Down below it is described the process required to follow to invoice an existing order.
